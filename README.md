# @raae/gatsby-theme-mui

> Adds MUI (mui.com) to your Gatsby site

## How to install

`npm install @raae/gatsby-theme-mui`

or

`yarn add @raae/gatsby-theme-mui`

## How to use

```
module.exports = {
  plugins: [
    `@raae/gatsby-theme-mui`
  ],
}
```
